# Countdown
Don't use this... but if you must..

This is a quick and dirty bar item `countdown` that shows the next event in the text file. I use it for F1. With the current dataset,

```
2026-05-01 16:30 🇺🇸 Miami GP: FP1
2026-05-01 20:30 🇺🇸 Miami GP: Sprint Qualy
2026-05-02 16:00 🇺🇸 Miami GP: Sprint
2026-05-02 20:00 🇺🇸 Miami GP: Qualy
2026-05-03 20:00 🇺🇸 Miami GP: Grand Prix
```

... the bar will show `🇺🇸 Miami GP: FP1: 4d 15h 28m`. 

I like to add the bar items `spacer,countdown` so this item is right-aligned in a top bar.
